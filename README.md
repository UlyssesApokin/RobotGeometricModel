# What is this?

Method for constructing geometric models of manipulation systems of industrial robots and multi-axis machines—é

# How to get?

````
$ git clone https://github.com/UlyssesApokin/RobotGeometricModel.git
````

````
$ cd RobotGeometricModel
````

````
$ gcc -Wall -g -ansi rgm2.c -lm -o rgm2
````

# How to use?

````
$ ./rgm2 robot1.rgm
````
